# Global Safe Companion

A modular AI-powered safety, monitoring, and intelligent assistance platform focused on secure automation, real-time analysis, and scalable companion-system architecture.

---

## Overview

**Global Safe Companion** is designed as a flexible framework for building intelligent companion systems that combine:

* AI/ML integration
* Real-time monitoring
* Secure automation workflows
* Adaptive response systems
* Modular service architecture
* Cross-platform deployment support

The project aims to provide a foundation for developing advanced safety-oriented digital companions capable of assisting users, monitoring environments, processing data streams, and integrating with external APIs and AI services.

# Installation

## Clone Repository

```bash
git clone https://github.com/swd-dev00/global-safe-companion.git
cd global-safe-companion
```

---

## Install Dependencies

### Node.js

```bash
npm install
```

### Python

```bash
pip install -r requirements.txt
```

---

## Environment Configuration

Create a `.env` file:

```env
OPENAI_API_KEY=your_key_here
HUGGINGFACE_API_KEY=your_key_here
MONGO_URI=your_database_uri
PORT=3000
```

---

# Running the Project

## Development Mode

```bash
npm run dev
```

## Production Mode

```bash
npm start
```

---

# Docker

## Build Container

```bash
docker build -t global-safe-companion .
```

## Run Container

```bash
docker run -p 3000:3000 global-safe-companion
```
